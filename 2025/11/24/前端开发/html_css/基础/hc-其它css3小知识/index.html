一、CSS3其他知识

1. box-sizing

  • 作用：改变盒子模型的度量方式，决定设置的宽高是内容区域还是可见区域
  • 默认值：
    content−boxcontent-boxcontent**−bo**x
    （设置宽高仅作用于内容区域）
  • 常用值：
    • content-box：宽高计算不包含padding和border
    • border-box：宽高计算包含padding和border
  • 开发实践：建议全局设置*{box-sizing: border-box;}使宽高设置与设计稿测量方式一致
  • 原因：设计稿测量的是可见区域宽度，而默认
    content−boxcontent-boxcontent**−bo**x
    会导致实际渲染尺寸超出预期
2. 图像内容适应

  • 应用场景：控制多媒体内容（img/video）在固定尺寸容器中的显示方式
  • 核心属性：
    object−fitobject-fitobject**−fi**t
  • 属性值：
    • fill（默认）：拉伸填充整个容器，可能导致比例失调
    • contain：保持比例完整显示，可能留有空白
    • cover：保持比例完全覆盖，可能裁剪部分内容
    • none：无视容器尺寸，显示原始大小
  • 常用组合：通常配合固定宽高的img/video元素使用
  • 实际应用：封面图推荐使用cover，产品展示图推荐使用contain
3. 视口单位

  • 新增单位：CSS3新增了vw和vh单位，分别表示视口宽度(Viewport Width)和视口高度(Viewport Height)
  • 单位换算：
    • 1vh1vh1vh
      = 视口高度的1%
    • 100vw100vw100vw
      = 视口宽度的100%
  • 视口定义：视口指整个浏览器的可视窗口区域
  • 实际应用：
    • 设置width:100vw可使元素宽度与视口宽度相同
    • 设置height:100vh可使元素高度与视口高度相同
  • 特性验证：
    • 元素设置100vh时，高度会完全撑满视口，可能产生滚动条
    • 元素设置100vw时，宽度会完全撑满视口宽度
  • 单位记忆：
    • v = viewport(视口)
    • w = width(宽度)
    • h = height(高度)
  • 兼容性：该单位是CSS3新增特性，CSS2中不存在
4. 伪元素选择器

  • 基本概念：通过::before和::after选择器，可以在不修改HTML结构的情况下为元素生成两个虚拟子元素
  • 应用场景：
    • 适合用于纯样式元素（如箭头、小三角等非文档内容）
    • 避免在HTML中使用过多无意义的空元素
  • 核心特性：
    • 必须设置content属性才能生效，无内容时可设为空字符串content: ‘’
    • 生成位置：::before插入在元素内容前，::after插入在元素内容后

5. 平滑滚动

  • 实现方式：通过CSS属性scroll-behavior: smooth实现
  • 特性对比：
    • auto：默认值，瞬间跳转到目标位置
    • smooth：产生平滑过渡的滚动效果
  • 应用优势：
    • 替代传统JS实现的滚动动画
    • 兼容现有滚动行为（锚点跳转、CSSOM scrolling API等）
  • 浏览器支持：
    • 适用于滚动容器元素
    • 在根元素设置时会影响视窗滚动

  • 注意事项：
    • 仅影响程序触发的滚动（用户手动滚动不受影响）
    • 动画类型为离散值（discrete），不支持过渡效果
6. 字体图标
1）font-face指令介绍

  • 指令作用：CSS3新增的@font-face指令允许加载网络字体或本地字体，解决了传统网页字体依赖用户本地安装的问题。
  • 工作原理：先通过local()查找本地字体，若未找到则通过url()下载网络字体资源。
  • 语法示例：
  • 应用场景：最常见于字体图标实现，也可用于加载特殊设计字体。
  • 兼容性说明：该规则既可用于CSS顶层，也可用于任何CSS条件组规则中。
2）字体图标的应用

  • 本质特征：字体图标本质是特殊设计的文字，可通过color设置颜色，font-size调整尺寸。
  • 技术实现：通过@font-face加载包含图标的字体文件，使用时像普通文字一样引用。
  • 优势对比：
    • 传统图片：体积大、修改颜色需重新制作
    • 字体图标：矢量无损缩放、通过CSS控制样式
  • 实际案例：将”你好，html”文字替换为图标字体显示效果。
3）字体图标平台介绍

  • 主流平台：国内最常用的是阿里巴巴矢量图标库(iconfont.cn)，提供海量免费图标资源。
  • 使用流程：
    • 登录平台搜索所需图标
    • 获取字体文件或CSS引用代码
    • 通过@font-face引入项目
  • 资源格式：支持SVG、AI、PNG等多种格式下载，但推荐使用字体形式。
  • 行业现状：已成为现代网页设计的标配，基本取代了小图标图片的使用。
7. 应用案例
1）例题：字体图标使用

  • 国内主流平台：阿里巴巴矢量图标库(iconfont.cn)是国内使用最广泛的字体图标平台
  • 使用步骤：
    • 注册会员账号
    • 创建项目（如”淘宝网”项目）
    • 搜索并添加所需图标到项目
    • 更新项目生成CSS链接
    • 在HTML中通过 <link>引入CSS文件
  • 图标管理：
    • 通过购物车批量管理图标
    • 添加至指定项目后需点击”更新”
    • 复制生成的CSS链接代码
  • 实现原理：
    • 本质是通过伪元素::before的content属性显示特殊字符
    • 使用@font-face指令加载网络字体
    • 通过font-size调整大小，color改变颜色
  • 本地使用方式：
    • 下载图标压缩包到本地
    • 解压后复制所有文件到项目目录
    • 引用本地的iconfont.css文件
    • 更新图标时需要重新下载覆盖文件
2）例题：字体图标练习

  • 练习要点：
    • 实现页面平滑滚动效果
    • 使用字体图标作为导航元素
    • 通过CSS控制图标大小和颜色
    • 结合伪元素实现特殊效果
  • 注意事项：
    • 在线使用时需添加http://协议前缀
    • 本地使用时需保持字体文件路径正确
    • 更新图标后需要重新生成CSS
    • 多色图标需要开启彩色选项

二、错误箭头制作

1. 基础线条制作
  • 本质构成：错误箭头由一根直线和一个小三角组成
  • 线条设置：
    • 高度设置为2像素
    • 宽度设置为400像素
    • 背景颜色设置为黑色（#000）
    • 通过CSS样式直接创建基础线条
2. 箭头三角制作
  • 伪元素应用：使用::before伪元素创建三角，避免额外HTML元素
    • 必须设置content属性（即使为空）
    • 默认display属性为inline，需改为block
  • 三角样式：
    • 宽高设为0
    • 通过border属性创建：
    • 保留需要的边框颜色（如左边框），其他设为transparent
3. 定位技巧
  • 父子定位：采用”子绝父相”原则
    • 父元素设置position: relative
    • 伪元素设置position: absolute
  • 位置调整：
    • right值设为-20px（边框宽度10px的2倍）
    • top值微调至-9px左右
    • 通过负值实现精确定位
4. 斜箭头实现
  • 差异化处理：
    • 将其中一个边框（如左边框）加宽至15px
    • 使用transform: rotate(130deg)旋转
  • 位置微调：
    • left设为50%
    • top值通过负值调整（约-11px）
    • 多次试验找到最佳角度和位置

三、课程总结

  • 核心要点：
    • 掌握伪元素创建装饰性元素的方法
    • 理解CSS边框实现三角的技巧
    • 熟练运用绝对定位实现元素精确定位
  • 实践建议：
    • 完成课堂练习巩固知识点
    • 尝试调整参数观察不同效果
    • 保存好课程资料供日后参考

四、知识小结

知识点 核心内容 考试重点/易混淆点 难度系数
box-sizing 改变盒子模型度量方式，content-box(默认)设置内容区宽高，border-box设置可见区域宽高 实际开发必用技巧：全局设置* { box-sizing: border-box } ⭐⭐
object-fit 控制多媒体内容适应方式，含fill(拉伸填充)、contain(保持比例)、cover(比例覆盖)、none(原始尺寸) 图片比例控制关键属性，与background-size区别 ⭐⭐
视口单位(vw/vh) 新增CSS3单位，1vw=视口宽度1%，100vh等于完整视口高度 替代百分比布局，移动端适配核心单位 ⭐⭐
伪元素选择器 ::before/::after动态生成子元素，需配合content属性使用 与伪类区别：单冒号(:)vs双冒号(::) ⭐⭐⭐
平滑滚动 scroll-behavior: smooth实现锚点平滑滚动，替代JS滚动动画 浏览器兼容性注意点 ⭐
@font-face 网络字体加载指令，突破用户本地字体限制 字体图标技术基础，WOFF格式兼容性 ⭐⭐⭐
字体图标 通过iconfont.cn平台使用，本质是特殊字体文件 开发规范：优先使用 <i class="iconfont">标签 ⭐⭐
弹性布局实战 flex-direction: column+justify-content/align-items实现完美居中 主轴方向变化时的属性适应 ⭐⭐⭐
CSS过渡动画 transition配合transform实现缩放/旋转效果 性能优化：优先使用transform和opacity ⭐⭐
绝对定位技巧 子绝父相定位配合translate实现精准定位 百分比定位与transform的差异 ⭐⭐⭐
