js-文字滚动效果

一、文字滚动效果源码

1. 编写HTML
1）引用CSS文件

  • 引用方式: 在 <head>标签内使用 <link>标签引用CSS文件，语法为 <link rel="stylesheet" href="路径">
  • 注意事项: CSS文件路径可以是相对路径或绝对路径，建议使用相对路径便于项目迁移
2）引用JavaScript文件

  • 两种引用方式:
    • 方式1：放在 <body>标签最底部，确保DOM加载完成后再执行JS
    • 方式2：放在 <head>标签内但添加defer属性，使JS异步加载不阻塞页面渲染
  • defer属性: 让脚本延迟到文档解析完成后再执行，避免阻塞页面加载
3）HTML结构书写

  • 容器设计: 使用 <div class="container">作为外层容器
  • 标题元素: 推荐使用 <h1 class="title">作为标题标签
  • 列表结构: 使用 <ul class="list">配合 <li>实现列表项
  • 开发技巧: 静态页面开发时不必过于关注内容细节，重点在于结构搭建和样式实现
4）使用乱数假文填充内容

  • 定义: 乱数假文(Lorem ipsum)是排版设计常用的无意义填充文本
  • 使用方法: 在HTML中直接输入”lorem”后跟数字(如lorem4)可生成指定单词数的假文
  • 应用场景:
    • 页面开发初期快速填充内容
    • 测试文字排版效果
    • 演示页面布局
  • 优势: 避免在开发阶段纠结具体内容，专注于样式实现
  • 生成控制: 数字参数控制生成单词数量，如”lorem4”生成4个单词的段落
  • 实际应用: 在 <li>标签中使用lorem生成4个列表项，每项包含4个单词的假文
  • 注意事项: 乱数假文仅用于开发测试，正式项目需替换为真实内容
2. 编写CSS
1）设置背景颜色

  • 背景色选择：使用取色器工具选取浅蓝色背景
    #bfedfc
    ，实际项目中可灵活调整颜色值
  • 容器设置：初始不设置固定高度，通过内部元素控制自适应高度
2）容器高度与元素排列

  • 布局方案：采用浮动(float)实现标题与列表的并排排列
  • 浮动设置：.title和.list均设置float: left
  • 高度问题：未清除浮动时容器高度显示为0，出现高度坍塌现象
3）解决高度坍塌问题

  • 清除浮动方法：
    • 方案1：使用伪元素清除法.container:after{content:’’;display:block;clear:both}
    • 方案2：设置容器overflow:hidden触发BFC
  • 最终选择：采用overflow:hidden方案，同时可处理内容溢出情况
4）标题样式设置

  • 字体设置：
    • 字号：固定值
      16px16px16px
    • 字重：font-weight:normal取消加粗效果
  • 间距控制：
    • 左外边距：
      20px20px20px
    • 上外边距：
      5px5px5px
      实现垂直对齐
    • 清除默认margin：margin:0重置浏览器默认样式
5）列表样式与滚动条

  • 基础样式：
    • 清除列表符号：list-style:none
    • 清除内边距：padding:0
  • 尺寸控制：
    • 固定高度：
      30px30px30px
    • 溢出处理：overflow:hidden scroll实现垂直滚动
  • 调试技巧：临时添加边框辅助调试，完成后移除
6）列表项高度与文字居中

  • 高度控制：每个 <li>设置固定高度
    30px30px30px
  • 垂直居中：通过line-height:30px实现单行文字垂直居中
  • 间距调整：列表整体设置margin-left:50px与标题保持距离
7）调整边距与对齐

  • 默认样式重置：
    • 清除 <ul>默认margin
    • 清除 <h1>默认margin
  • 精细调整：
    • 标题上边距：
      5px5px5px
      实现视觉对齐
    • 列表左外边距：
      50px50px50px
      保持合适间距
8）添加竖线边框

  • 实现方法：利用 <h1>的右边框border-right
  • 样式细节：
    • 宽度：
      2px2px2px
    • 类型：solid
    • 颜色：浅灰色
      #ccc
  • 间距控制：通过padding-right:30px调整竖线与文字间距
9）容器最终样式

  • 内边距设置：
    • 上下：
      20px20px20px
    • 左右：
      000
  • 完整样式：padding:20px 0实现内容区域呼吸空间
3. 编写JavaScript
1）网页分析

  • 核心逻辑: 网页开发主要考虑两件事：初始化和交互
  • 初始化定义: 网页加载后自动执行的操作，无需用户触发
  • 交互定义: 用户操作后触发的网页响应行为
2）初始化与交互
  • 开发流程:
    • 先处理初始化：确定页面加载时需要完成的工作
    • 再处理交互：定义用户可能的操作及对应响应
  • 典型场景: 当前案例只有初始化需求，无用户交互
3）无交互效果
  • 特点: 纯自动滚动效果，用户操作不会影响页面行为
  • 实现重点: 只需专注初始化阶段的自动滚动逻辑
4）立即执行函数

  • 作用:
    • 避免全局变量污染
    • 防止命名冲突
  • 语法: (function(){…})()
  • 现代替代: 工程化项目中有更好的模块化方案
5）滚动效果实现
  • 无缝滚动原理
    • 关键技术: 克隆首项到列表末尾
    • 视觉原理: 滚动到最后项时瞬间跳回首项，利用相同内容制造无限滚动假象
    • 实现步骤:
      • 获取列表元素
      • 克隆第一个子元素
      • 将克隆元素追加到列表末尾
  • 克隆实现细节
    • 元素获取:
      • 使用document.querySelector(‘.list’)获取列表
      • 通过children[0]访问首项
    • 克隆方法:
      • cloneNode(true)进行深度克隆（包含子元素）
      • appendChild()将克隆元素添加到列表末尾
  • 定时滚动机制
    • 避免硬编码:
      • 使用duration变量存储间隔时间（2000ms）
      • 定义curIndex记录当前显示项索引
    • 滚动计算:
      • 下一项位置：curIndex++
      • 滚动高度：to = curIndex * itemHeight（每项高度30px）
    • 执行方式:
      • 通过setInterval(moveNext, duration)定时触发
      • 直接设置list.scrollTop = to改变滚动位置
6）列表元素克隆
  • 注意事项:
    • 必须使用cloneNode(true)进行深度克隆
    • 克隆后需要通过appendChild添加到DOM才能生效
  • 调试技巧:
    • 通过控制台直接修改scrollTop验证滚动效果
    • 输出curIndex观察滚动状态变化
7）滚动动画实现

  • 基础实现原理：
    • 通过cloneNode复制首项并追加到列表末尾实现循环滚动
    • 使用setInterval定时调用moveNext函数实现自动滚动
    • 核心变量：itemHeight（项高度）、curIndex（当前项索引）
  • 动画本质：
    • 不是瞬间到达目标位置，而是通过逐渐改变scrollTop值实现平滑过渡
    • 需要计算起始值（from）和目标值（to）：
      • from = curIndex * itemHeight
      • to = (curIndex + 1) * itemHeight
8）动画参数设置

  • 关键参数：
    • 总时间：完成整个变化的时间（示例使用500ms）
    • 间隔时间：每次变化的时间间隔（建议10-15ms）
    • 变化次数：总时间/间隔时间（500/10=50次）
    • 单次变化量：(to - from)/变化次数
  • 实现步骤：
    • 计算起始值from和目标值to
    • 设置setInterval定时器，间隔时间为设定的小间隔
    • 每次执行时：from += 单次变化量
    • 当from >= to时清除定时器
  • 调试技巧：
    • 通过console.log输出from值观察变化过程
    • 调整总时间可改变动画速度（如300ms会更快）
    • 注意浮点数运算可能产生微小误差（如70.199999）
  • 注意事项：
    • 间隔时间越短动画越平滑（但性能消耗越大）
    • 必须及时清除已完成动画的定时器
    • 实际开发中建议使用CSS动画或现成动画库提高效率
9）滚动到最后一项处理
  • 判断最后一项逻辑
    • 判断条件：通过比较当前索引
      curIndexcurIndexcurIndex
      与列表子元素最大下标list.children.length−1list.children.length-1list**.children**.length**−**1
      是否相等来判断是否到达最后一项
    • 索引机制：
      curIndexcurIndexcurIndex
      从0开始计数，0表示第一项，1表示第二项，依此类推
    • 子元素获取：使用DOM的
      childrenchildrenchildren
      属性获取列表所有子元素，lengthlengthlength
      属性获取子元素数量
  • 最后一项处理方案
    • 重置位置：当滚动到最后一项时，需要将滚动位置重置为0，即
      list.scrollTop=0list.scrollTop = 0list**.scrollTop**=0
    • 索引归零：必须同时将
      curIndexcurIndexcurIndex
      重置为0，否则会导致后续计算的目标值异常增大
    • 两种实现方式：
      • 方法一：在判断为最后一项后直接结束函数执行
      • 方法二：将
        fromfromfrom
        变量设置为0，让后续代码自然处理滚动
  • 完整实现流程
    • 滚动动画：
      • 使用
        setIntervalsetIntervalsetInterval
        实现分步滚动效果
      • totalDurationtotalDurationtotalDuration
        控制动画总时间(300ms)
      • durationdurationduration
        控制每次变化间隔(10ms)
      • 通过计算每次变化量
        disdisdis
        实现平滑滚动
    • 边界处理：
      • 到达目标高度后清除定时器
        clearInterval(timerId)clearInterval(timerId)clearInterval**(timerId)**
      • 特别处理最后一项情况，实现循环滚动效果
    • 调试技巧：
      • 使用
        console.logconsole.logconsole.log
        输出调试信息
      • 观察
        fromfromfrom
        值的变化过程验证逻辑正确性
4. 文字滚动效果实现

1）CSS样式设置
  • 容器设置:
    • overflow: hidden 隐藏超出容器范围的内容
    • background: #bfedfc 设置浅蓝色背景
    • padding: 20px 0 设置上下内边距为20px
  • 标题样式:
    • float: left 标题左浮动
    • font-size: 16px 设置字体大小
    • border-right: 2px solid #ccc 右侧添加分隔线
    • padding-right: 30px 标题右侧留白
  • 列表样式:
    • height: 30px 固定列表高度
    • overflow: hidden 隐藏溢出内容
    • margin-left: 50px 左侧外边距
2）JavaScript实现逻辑

  • 初始化阶段
    • 元素克隆:
      • 使用cloneNode(true)方法克隆列表第一个元素
      • 将克隆的元素追加到列表末尾，实现无缝滚动基础
    • 定时器设置:
      • 设置duration=2000表示每2秒滚动一次
      • 通过setInterval(moveNext, duration)启动定时滚动
  • 滚动动画实现
    • 参数计算:
      • itemHeight=30 每项固定高度30px
      • curIndex 记录当前显示项索引
      • from/to 计算起始和目标滚动位置
    • 平滑动画:
      • totalDuration=300 动画总时长300ms
      • duration=10 每10ms变化一次
      • dis=(to-from)/times 计算每次变化量
    • 边界处理:
      • 当from >= to时清除定时器
      • 通过clearInterval(timerId)停止动画
3）实现要点
  • 训练目的:
    • 重点不在于效果本身的重要性，而是通过实现过程培养解决问题的能力
    • 这种能力需要长期训练积累，是开发者的核心素质
  • 代码思路:
    • 先完成元素克隆建立循环基础
    • 再实现定时滚动功能
    • 最后添加平滑动画效果提升用户体验
  • 实际应用:
    • 类似效果可用于公告栏、新闻轮播等场景
    • 核心思路可迁移到其他需要动态展示的界面元素
