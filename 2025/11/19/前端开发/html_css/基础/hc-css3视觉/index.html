一、课前概述

1. 前端开发三大语言
  • 核心基础: 前端开发主要基于HTML、CSS和JavaScript三大语言，所有高级框架和工具都建立在这三个基础之上。
  • 版本演进: 当前主流版本为HTML5、CSS3和ES2021，课程重点讲解这些新版本带来的变化。
2. 课程学习内容与版本更新
  • 学习重点: 对比旧版本(HTML4、CSS2、ES3)与新版本的区别，掌握新增特性。
  • 版本特点: 新版本完全兼容旧版本，在保留原有功能基础上增加了新特性。
3. 学习顺序与CSS3手册介绍

  • 学习顺序: 先讲CSS3(视觉→过渡动画→布局→其他)，再讲HTML5，最后是ES6到ES2021。
  • 手册特点:
    • 精选最常用核心功能，非完整官方文档
    • 特别设计为易理解版本，适合初学者查阅
    • 课程中将反复使用该手册进行演示
4. CSS3主要变化与学习路线
  • 三大模块:
    • 布局：提供浮动、弹性盒、网格等新方案
    • 视觉：不影响布局的样式效果
    • 过渡和动画：实现动态效果
  • 教学重点: 掌握手册使用方法比死记硬背更重要

二、视觉类样式

1. 视觉样式定义
  • 核心特征: 只改变元素视觉效果，不影响盒子模型的位置、尺寸和布局。
  • 典型示例: 文字颜色、背景颜色、圆角、阴影等属性。
2. 盒子阴影

  • 基本语法: box-shadow: h-shadow v-shadow blur-radius color
  • 参数说明:
    • 偏移量: 水平(h-shadow)和垂直(v-shadow)偏移，正负值控制方向
    • 模糊半径: 控制阴影模糊程度，值越大越模糊
    • 颜色值: 支持颜色名称、十六进制或RGBA格式
  • 实际示例:
    • box-shadow: 8px 15px teal → 右下方teal色阴影
    • box-shadow: 8px -15px rgba(0,0,0,0.5) → 右上方半透明黑色阴影
    • box-shadow: 8px 15px 5px teal → 带5px模糊半径的阴影
    • box-shadow: 8px 15px 10px rgba(0,0,0,0.5) → 大模糊半径的半透明阴影
  • 学习建议: 通过MDN文档和手册中的示例进行实践练习，理解各参数的实际效果。

三、视觉样式

1. 视觉类的CSS3
1）盒子阴影
  • box-shadow属性：通过该属性可以设置整个盒子的阴影效果
  • 参数说明：
    • 水平偏移：正数向右，负数向左（如8px或-8px）
    • 垂直偏移：正数向下，负数向上（如15px或-15px）
    • 模糊半径：值越大模糊范围越大（如5px或10px）
    • 颜色值：支持颜色名称（如teal）或RGBA格式（如rgba(0,0,0,0.5)）
  • 特殊用法：
    • inset：设置阴影向内扩散
    • 多阴影：用逗号分隔多个阴影声明，前者会覆盖后者
2）文字阴影
  • text-shadow属性：专门用于设置文字阴影效果
  • 参数说明：
    • 前两个参数为水平和垂直偏移
    • 第三个参数为模糊半径
    • 最后一个参数为阴影颜色
  • 特殊用法：
    • 多阴影叠加：通过设置多个偏移方向相同的阴影可以实现文字描边效果
    • 颜色继承：不指定颜色时会默认使用文字颜色
3）圆角

  • border-radius属性：设置盒子四个角的圆角程度
  • 常用值：
    • 固定值：如10px，四个角统一设置
    • 百分比：50%可将方形元素变为圆形
    • 多值设置：分别设置左上、右上、右下、左下四个角（如10px 20px 30px 40px）
  • 实现原理：用指定半径的圆形切割元素的直角
4）背景渐变

  • linear-gradient()函数：创建线性渐变背景图像
  • 参数说明：
    • 第一个参数：渐变方向（to top/bottom/left/right）
    • 后续参数：颜色节点（如#e66465到#9198e5）
  • 特点：可以设置多个颜色节点，实现复杂渐变效果
5）变形

  • translate平移

    • transform属性：使用translate实现元素位移
    • 特点：
      • 不影响文档流布局（类似相对定位）
      • 高性能（不会触发重排和重绘）
    • 参数形式：
      • translateX()：水平移动
      • translateY()：垂直移动
      • translate()：同时设置XY轴位移

  • scale缩放

    • 缩放类型：
      • scale()：整体缩放
      • scaleX()：水平缩放
      • scaleY()：垂直缩放
    • 参数说明：
      • 1表示原始大小
      • 大于1放大，小于1缩小
      • 支持负值（会产生镜像效果）

  • rotate旋转

    • 角度单位：
      • deg：度数（如45deg）
      • turn：圈数（如0.5turn=180deg）
    • 旋转方向：正值顺时针，负值逆时针
    • 坐标系影响：旋转会改变元素的局部坐标系

  • 改变变形原点

    • transform-origin属性：设置变形基准点
    • 常用值：
      • 关键词：left/top/right/bottom/center
      • 具体坐标值（如30px 60px）
    • 影响范围：所有变形操作（旋转、缩放等）都会基于该点进行

  • 多种变形叠加

    • 组合方式：在transform属性中用空格分隔多个变形函数
    • 执行顺序：从右向左依次应用
    • 注意事项：
      • 旋转会改变局部坐标系，影响后续平移方向
      • 不同顺序的组合会产生不同视觉效果

    • border-radius属性：
      • 用于设置盒子的圆角效果
      • 基本语法：border-radius: 值;
      • 可接受像素值(px)或百分比值(%)
    • 圆角实现方式：
      • border-radius:10px：同时设置4个角的圆角，半径为10px
      • border-radius:50%：同时设置4个角的圆角，圆的横向半径为宽度一半，纵向半径为高度一半
      • border-radius:10px 20px 30px 40px：分别设置左上、右上、右下、左下的圆角

    • 固定值实现：
      • 使用固定像素值：border-radius:50px
      • 适用于已知固定尺寸的元素
    • 百分比实现：
      • 使用百分比值：border-radius:50%
      • 自动计算宽度和高度的一半作为圆角半径
      • 适用于响应式设计，元素尺寸变化时仍保持圆形
    • 几何原理：
      • 当元素为正方形时，设置50%圆角会形成完美圆形
      • 原理：圆的半径等于元素宽度和高度的一半

      • 使盒子形态发生变化而不影响布局

      • 使用rotate()函数进行旋转

      • 使用负margin调整：margin-left: -宽度一半，margin-top: -高度一半
    • 现代居中方法：
      • 使用transform: translate(-50%, -50%)
      • 不需要知道盒子宽高
      • 先设置left:50%和top:50%
      • 再通过translate(-50%,-50%)将盒子向左和上各移动自身尺寸的50%

  • 开发学习建议: 避免死记硬背，通过实际项目反复练习掌握属性用法

五、知识小结

知识点 核心内容 考试重点/易混淆点 难度系数
CSS3视觉类属性 阴影（box-shadow）、圆角（border-radius）、背景渐变（linear-gradient）、变形（transform） 多阴影叠加（顺序覆盖）、圆角百分比计算（50%生成圆形） ⭐⭐
变形（Transform） 平移（translate）、缩放（scale）、旋转（rotate）、变形原点（transform-origin） 旋转中心点调整（需结合坐标计算）、多变形叠加顺序影响结果 ⭐⭐⭐
绝对/固定定位居中 transform: translate(-50%, -50%)无需定宽高的居中方案 与传统margin负值方案对比 ⭐⭐
CSS三角形实现 零宽高+边框色透明技巧（border-color: transparent） 边框宽度与三角形大小的关系 ⭐
交互状态样式 :hover/:active伪类结合阴影与变形实现动态效果 多阴影叠加的视觉层次控制 ⭐⭐
文档驱动开发 强调查阅手册而非死记属性，提供核心概念+查阅路径 属性名记忆与实际应用优先级 ⭐
