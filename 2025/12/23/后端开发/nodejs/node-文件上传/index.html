    • ж–Үжң¬еҹҹеӨ„зҗҶпјҡ
      • .none()пјҡд»…еӨ„зҗҶж–Үжң¬еҹҹпјҢиӢҘж”¶еҲ°ж–Үд»¶дјҡжҠҘй”ҷ
      • .any()пјҡжҺҘеҸ—жүҖжңүдёҠдј ж–Үд»¶пјҲж…Һз”Ёпјү
    • ж–Үд»¶еҜ№иұЎеұһжҖ§пјҡ
      • fieldnameпјҡиЎЁеҚ•жҢҮе®ҡзҡ„еӯ—ж®өеҗҚ
      • originalnameпјҡз”ЁжҲ·еҺҹе§Ӣж–Үд»¶еҗҚ
      • encodingпјҡж–Үд»¶зј–з Ғ
      • mimetypeпјҡж–Үд»¶MIMEзұ»еһӢ
      • sizeпјҡж–Үд»¶еӨ§е°ҸпјҲеӯ—иҠӮпјү
      • destinationпјҡеӯҳеӮЁи·Ҝеҫ„пјҲDiskStorageпјү
      • filenameпјҡеӯҳеӮЁж–Үд»¶еҗҚпјҲDiskStorageпјү
      • pathпјҡе®Ңж•ҙж–Үд»¶и·Ҝеҫ„пјҲDiskStorageпјү
      • bufferпјҡж–Үд»¶BufferпјҲMemoryStorageпјү

  • й…ҚзҪ®ж–Үд»¶еӯҳеӮЁи·Ҝеҫ„

    • destй…ҚзҪ®пјҡ
      • жңҖеҹәжң¬й…ҚзҪ®йЎ№пјҢжҢҮе®ҡж–Үд»¶еӯҳеӮЁзӣ®еҪ•
      • зӨәдҫӢпјҡdest: path.resolve(__dirname, вҖң../../public/uploadвҖқ)
      • зӣ®еҪ•дёҚеӯҳеңЁж—¶дјҡиҮӘеҠЁеҲӣе»ә
    • жіЁж„ҸдәӢйЎ№пјҡ
      • жңӘй…ҚзҪ®destж—¶ж–Үд»¶дҝқеӯҳеңЁеҶ…еӯҳдёӯ
      • дёҠдј ж–Үд»¶дјҡиў«йҮҚе‘ҪеҗҚд»ҘйҒҝе…ҚеҶІзӘҒ
      • з”ҹдә§зҺҜеўғеә”й…ҚзҪ®.gitignoreеҝҪз•ҘдёҠдј зӣ®еҪ•
    • е…¶д»–й…ҚзҪ®йЎ№пјҡ
      • storageпјҡжӣҝд»Јdestзҡ„жӣҙй«ҳзә§еӯҳеӮЁй…ҚзҪ®
      • fileFilterпјҡж–Үд»¶зұ»еһӢиҝҮж»ӨеҷЁ
      • limitsпјҡйҷҗеҲ¶дёҠдј ж•°жҚ®еӨ§е°Ҹ
      • preservePathпјҡдҝқз•ҷе®Ңж•ҙж–Үд»¶и·Ҝеҫ„

дәҢгҖҒж–Үд»¶дёҠдј 

1. е®ўжҲ·з«Ҝж–Үд»¶дёҠдј 
1пјүж–Үд»¶дёҠдј дёҺеҗҺзјҖеҗҚй—®йўҳ
  • й»ҳи®ӨеӨ„зҗҶжңәеҲ¶пјҡжңҚеҠЎеҷЁй»ҳи®ӨдёҚдҝқеӯҳж–Үд»¶еҗҺзјҖеҗҚпјҢиҝҷжҳҜдёәдәҶе®үе…ЁиҖғиҷ‘пјҢеӣ дёәйңҖиҰҒеӨ„зҗҶеҗ„з§Қзұ»еһӢзҡ„ж–Үд»¶
  • еӣҫзүҮеӨ„зҗҶеңәжҷҜпјҡеңЁеӣҫзүҮдёҠдј еңәжҷҜдёӯйңҖиҰҒдҝқз•ҷеҗҺзјҖеҗҚпјҢеҗҰеҲҷдјҡеҜјиҮҙжөҸи§ҲеҷЁж— жі•жӯЈзЎ®иҜҶеҲ«ж–Үд»¶зұ»еһӢ
  • и§ЈеҶіж–№жЎҲпјҡеҸҜд»ҘйҖҡиҝҮй…ҚзҪ®еӯҳеӮЁеј•ж“ҺжқҘи§ЈеҶіеҗҺзјҖеҗҚй—®йўҳпјҢзЎ®дҝқж–Үд»¶иғҪжӯЈзЎ®жҳҫзӨә
2пјүе®ўжҲ·з«Ҝж–Үд»¶дёҠдј д»Јз ҒзӨәдҫӢ
  • FormDataжһ„е»әпјҡ
    • дҪҝз”Ёnew FormData()жһ„е»әиЎЁеҚ•ж•°жҚ®
    • йҖҡиҝҮappend()ж–№жі•ж·»еҠ ж–Үжң¬еӯ—ж®өе’Ңж–Үд»¶еӯ—ж®ө
    • ж–Үд»¶еӯ—ж®өйңҖиҰҒеҫӘзҺҜеӨ„зҗҶinpFile.filesж•°з»„
  • fetchиҜ·жұӮпјҡ
    • и®ҫзҪ®иҜ·жұӮж–№жі•дёәPOST
    • иҜ·жұӮдҪ“зӣҙжҺҘдј е…ҘFormDataеҜ№иұЎ
    • еӨ„зҗҶе“Қеә”з»“жһңж—¶дҪҝз”Ё.text()ж–№жі•иҺ·еҸ–ж–Үжң¬е“Қеә”
3пјүж–Үд»¶дёҠдј и·Ҝеҫ„й—®йўҳ
  • и·Ҝеҫ„еј•з”Ёй—®йўҳпјҡ
    • дёҠдј еҗҺзҡ„ж–Үд»¶еӯҳеӮЁеңЁpublic/uploadзӣ®еҪ•дёӢ
    • йҖҡиҝҮ <img>ж Үзӯҫзҡ„srcеұһжҖ§еј•з”ЁдёҠдј ж–Үд»¶
    • ж— еҗҺзјҖеҗҚж–Үд»¶дјҡеҜјиҮҙжөҸи§ҲеҷЁж— жі•жӯЈзЎ®иҜҶеҲ«ж–Үд»¶зұ»еһӢ
  • йӘҢиҜҒж–№жі•пјҡ
    • зӣҙжҺҘеңЁйЎөйқўдёӯйҖҡиҝҮеӣҫзүҮе…ғзҙ еј•з”ЁдёҠдј ж–Үд»¶
    • и§ӮеҜҹжөҸи§ҲеҷЁжҳҜеҗҰиғҪжӯЈзЎ®жҳҫзӨәеӣҫзүҮеҶ…е®№
4пјүж–Үд»¶дёҠдј и·Ҝеҫ„иў«дҝ®ж”№зҡ„еҺҹеӣ 
  • historyдёӯй—ҙд»¶еҪұе“Қпјҡ
    • ж— еҗҺзјҖеҗҚж–Үд»¶иҜ·жұӮдјҡиў«йҮҚе®ҡеҗ‘еҲ°index.html
    • иҝҷжҳҜhistoryдёӯй—ҙд»¶зҡ„й»ҳи®ӨиЎҢдёә
    • дёҙж—¶и§ЈеҶіж–№жЎҲжҳҜжіЁйҮҠжҺүhistoryдёӯй—ҙд»¶
  • ж №жң¬и§ЈеҶіж–№жЎҲпјҡ
    • еҝ…йЎ»дҝқз•ҷж–Үд»¶еҗҺзјҖеҗҚ
    • зЎ®дҝқжөҸи§ҲеҷЁиғҪжӯЈзЎ®иҜҶеҲ«ж–Үд»¶зұ»еһӢ
    • йҒҝе…ҚиҜ·жұӮиў«й”ҷиҜҜйҮҚе®ҡеҗ‘
5пјүзЈҒзӣҳеӯҳеӮЁеј•ж“Һй…ҚзҪ®
  • еӯҳеӮЁеј•ж“Һзұ»еһӢпјҡ
    • зЈҒзӣҳеӯҳеӮЁ(DiskStorage)пјҡж–Үд»¶дҝқеӯҳеҲ°жңҚеҠЎеҷЁзЈҒзӣҳ
    • еҶ…еӯҳеӯҳеӮЁпјҡж–Үд»¶дҝқеӯҳеңЁеҶ…еӯҳдёӯ
    • з¬¬дёүж–№еӯҳеӮЁпјҡйңҖиҰҒиҮӘиЎҢе®һзҺ°
  • зЈҒзӣҳеӯҳеӮЁй…ҚзҪ®пјҡ
    • йҖҡиҝҮmulter.diskStorage()й…ҚзҪ®
    • destinationи®ҫзҪ®еӯҳеӮЁзӣ®еҪ•
    • filenameи®ҫзҪ®ж–Үд»¶еҗҚз”ҹжҲҗи§„еҲҷ
  • е®үе…ЁиӯҰе‘Ҡпјҡ
    • дёҚиҰҒе°ҶmulterдҪңдёәе…ЁеұҖдёӯй—ҙд»¶дҪҝз”Ё
    • еҸӘеңЁйңҖиҰҒеӨ„зҗҶж–Үд»¶дёҠдј зҡ„и·Ҝз”ұдёӯдҪҝз”Ё
    • йҳІжӯўжҒ¶ж„Ҹз”ЁжҲ·дёҠдј ж–Үд»¶еҲ°жңӘеҸ—дҝқжҠӨзҡ„и·Ҝз”ұ
2. жңҚеҠЎеҷЁж–Үд»¶дёҠдј 
1пјүupload.jsжәҗз Ғпјҡй…ҚзҪ®storage
  • еҹәжң¬з»“жһ„пјҡдҪҝз”Ёmulter.diskStorage()ж–№жі•еҲӣе»әеӯҳеӮЁеј•ж“ҺпјҢеҢ…еҗ«destinationе’ҢfilenameдёӨдёӘе…ій”®й…ҚзҪ®йЎ№
  • з®ҖеҶҷж–№ејҸпјҡеҪ“еҸҳйҮҸеҗҚзӣёеҗҢж—¶еҸҜйҮҮз”ЁES6еұһжҖ§з®ҖеҶҷпјҢеҰӮmulter({ storage })
  • еӯҳеӮЁжҺ§еҲ¶пјҡзӣёжҜ”зӣҙжҺҘдҪҝз”ЁdestеҸӮж•°пјҢstorageй…ҚзҪ®иғҪжӣҙзІҫз»ҶжҺ§еҲ¶ж–Үд»¶еӯҳеӮЁдҪҚзҪ®е’Ңе‘ҪеҗҚи§„еҲҷ
2пјүзЈҒзӣҳеӯҳеӮЁеј•ж“Һпјҡdestinationе’Ңfilename
  • destinationеҮҪж•°пјҡ
    • дҪңз”ЁпјҡзЎ®е®ҡж–Үд»¶еӯҳеӮЁзӣ®еҪ•пјҢжҺҘж”¶req, file, cbдёүдёӘеҸӮж•°
    • еҲӣе»әиҙЈд»»пјҡдҪҝз”ЁеҮҪж•°еҪўејҸж—¶йңҖиҰҒжүӢеҠЁеҲӣе»әзӣ®еҪ•пјҢеӯ—з¬ҰдёІеҪўејҸеҲҷз”ұmulterиҮӘеҠЁеҲӣе»ә
    • й»ҳи®ӨеҖјпјҡжңӘи®ҫзҪ®ж—¶дҪҝз”Ёзі»з»ҹдёҙж—¶ж–Үд»¶еӨ№
  • filenameеҮҪж•°пјҡ
    • е‘ҪеҗҚи§„еҲҷпјҡйңҖиҝ”еӣһе®Ңж•ҙж–Үд»¶еҗҚпјҲеҗ«жү©еұ•еҗҚпјүпјҢmulterдёҚдјҡиҮӘеҠЁж·»еҠ жү©еұ•еҗҚ
    • й»ҳи®ӨиЎҢдёәпјҡжңӘи®ҫзҪ®ж—¶з”ҹжҲҗйҡҸжңәж— жү©еұ•еҗҚж–Үд»¶еҗҚ
    • еҸӮж•°иҜҙжҳҺпјҡreq.bodyеҸҜиғҪжңӘе®Ңе…ЁеЎ«е……пјҢйңҖжіЁж„Ҹеӯ—ж®өйЎәеәҸеҪұе“Қ
3пјүupload.jsжәҗз Ғпјҡdestinationи®ҫзҪ®
  • и·Ҝеҫ„жһ„е»әпјҡ
    • ж–№жі•пјҡдҪҝз”Ёpath.resolve(__dirname, вҖң../public/uploadвҖқ)жһ„е»әз»қеҜ№и·Ҝеҫ„
    • еұӮзә§е…ізі»пјҡйҖҡиҝҮ../иҝ”еӣһдёҠзә§зӣ®еҪ•пјҢеҶҚе®ҡдҪҚеҲ°public/uploadеӯҗзӣ®еҪ•
  • еӣһи°ғи§„иҢғпјҡ
    • й”ҷиҜҜеӨ„зҗҶпјҡз¬¬дёҖдёӘеҸӮж•°дј nullиЎЁзӨәж— й”ҷиҜҜ
    • и·Ҝеҫ„иҝ”еӣһпјҡз¬¬дәҢдёӘеҸӮж•°дј йҖ’жңҖз»ҲеӯҳеӮЁи·Ҝеҫ„еӯ—з¬ҰдёІ
  • жіЁж„ҸдәӢйЎ№пјҡйңҖзЎ®дҝқзӣ®ж Үзӣ®еҪ•еӯҳеңЁпјҢеҗҰеҲҷйңҖжүӢеҠЁеҲӣе»ә
4пјүupload.jsжәҗз Ғпјҡжһ„е»әз»қеҜ№и·Ҝеҫ„
  • и·Ҝеҫ„и§Јжһҗпјҡ
    • __dirnameиЎЁзӨәеҪ“еүҚжЁЎеқ—зӣ®еҪ•
    • path.resolve()ж–№жі•е°ҶзӣёеҜ№и·Ҝеҫ„и§Јжһҗдёәз»қеҜ№и·Ҝеҫ„
  • зӣ®еҪ•з»“жһ„пјҡ
    • зӨәдҫӢдёӯйҮҮз”ЁвҖқ../../вҖңи·Ёи¶ҠдёӨзә§зӣ®еҪ•
    • жңҖз»Ҳе®ҡдҪҚеҲ°йЎ№зӣ®ж №зӣ®еҪ•дёӢзҡ„public/uploadж–Үд»¶еӨ№
5пјүж–Үд»¶дҝЎжҒҜ
  • ж ёеҝғеӯ—ж®өпјҡ
    • fieldnameпјҡиЎЁеҚ•дёӯе®ҡд№үзҡ„еӯ—ж®өеҗҚ
    • originalnameпјҡе®ўжҲ·з«ҜеҺҹе§Ӣж–Үд»¶еҗҚ
    • mimetypeпјҡж–Үд»¶MIMEзұ»еһӢпјҲеҰӮimage/jpegпјү
    • sizeпјҡж–Үд»¶еӯ—иҠӮеӨ§е°Ҹ
  • еӯҳеӮЁзӣёе…іпјҡ
    • destinationпјҡзЈҒзӣҳеӯҳеӮЁзӣ®еҪ•пјҲд»…DiskStorageпјү
    • filenameпјҡеӯҳеӮЁеҗҺзҡ„ж–Үд»¶еҗҚ
    • pathпјҡж–Үд»¶е®Ңж•ҙеӯҳеӮЁи·Ҝеҫ„
6пјүupload.jsжәҗз Ғпјҡfilenameи®ҫзҪ®
  • еҹәзЎҖж–№жЎҲпјҡ
    • з»„еҗҲfile.fieldnameе’ҢDate.now()з”ҹжҲҗж–Үд»¶еҗҚ
    • зӨәдҫӢпјҡfile.fieldname + вҖң-вҖң + Date.now()
  • жү©еұ•дҝЎжҒҜпјҡ
    • еҸҜйҖҡиҝҮfile.originalnameиҺ·еҸ–е®ўжҲ·з«ҜеҺҹе§Ӣж–Үд»¶еҗҚ
    • дҪҝз”Ёpath.extname()жҸҗеҸ–ж–Үд»¶жү©еұ•еҗҚ
7пјүж–Үд»¶еҗҚжһ„е»әпјҡж—¶й—ҙжҲі+йҡҸжңәеӯ—з¬Ұ
  • е‘ҪеҗҚзӯ–з•Ҙпјҡ
    • ж—¶й—ҙжҲіпјҡдҪҝз”ЁDate.now()иҺ·еҸ–жҜ«з§’зә§ж—¶й—ҙжҲі
    • йҡҸжңәдёІпјҡMath.random().toString(36).slice(-6)з”ҹжҲҗ6дҪҚйҡҸжңәеӯ—з¬Ұ
    • жү©еұ•еҗҚпјҡд»Һfile.originalnameжҸҗеҸ–еҺҹе§Ӣжү©еұ•еҗҚ
  • е®Ңж•ҙзӨәдҫӢпјҡ
  • е“Қеә”еӨ„зҗҶпјҡ
    • иҝ”еӣһJSONж јејҸе“Қеә”еҢ…еҗ«ж–Үд»¶и®ҝй—®и·Ҝеҫ„
    • и·Ҝеҫ„ж јејҸпјҡ/upload/з”ҹжҲҗзҡ„ж–Үд»¶еҗҚ
3. ж–Үд»¶дёҠдј й…ҚзҪ®
1пјүж–Үд»¶еӨ§е°ҸйҷҗеҲ¶й…ҚзҪ®
  • limitsй…ҚзҪ®йЎ№пјҡз”ЁдәҺйҷҗеҲ¶дёҠдј ж–Үд»¶зҡ„еҗ„йЎ№еҸӮж•°пјҢеҢ…жӢ¬ж–Үд»¶еҗҚй•ҝеәҰгҖҒеӯ—ж®өеҖјй•ҝеәҰзӯү
  • fileSizeи®ҫзҪ®пјҡ
    • й»ҳи®ӨеҖјдёәinfinityпјҲж— йҷҗеҲ¶пјү
    • йңҖи®ҫзҪ®дёәж•ҙж•°пјҢеҚ•дҪҚжҳҜеӯ—иҠӮ
    • зӨәдҫӢпјҡйҷҗеҲ¶дёә10KBеҸҜи®ҫзҪ®дёә10*1024
  • й”ҷиҜҜеӨ„зҗҶпјҡ
    • и¶…иҝҮйҷҗеҲ¶дјҡжҠӣеҮәMulterError
    • й”ҷиҜҜж¶ҲжҒҜдёәвҖқfile too largeвҖқ
    • е»әи®®еңЁжңҚеҠЎеҷЁз«Ҝе’Ңе®ўжҲ·з«ҜйғҪиҝӣиЎҢйӘҢиҜҒ
2пјүж–Үд»¶зұ»еһӢиҝҮж»Ө
  • fileFilterй…ҚзҪ®пјҡ
    • жҺҘж”¶reqе’ҢfileеҜ№иұЎдҪңдёәеҸӮж•°
    • йҖҡиҝҮеҲӨж–ӯfile.originalnameиҺ·еҸ–ж–Үд»¶жү©еұ•еҗҚ
  • зҷҪеҗҚеҚ•жңәеҲ¶пјҡ
    • еҸҜи®ҫзҪ®е…Ғи®ёзҡ„еҗҺзјҖеҗҚж•°з»„пјҲеҰӮ[вҖң.jpgвҖқ, вҖң.gifвҖқ, вҖң.pngвҖқ]пјү
    • дёҚеңЁзҷҪеҗҚеҚ•еҲҷжҠӣеҮәй”ҷиҜҜдҝЎжҒҜвҖқyour extname is not supportвҖқ
